ScaledaReloadTasksAction

class ScaledaReloadTasksAction extends AnAction

Action: Load Scaleda project config, with all its targets and tasks. This action should be performed when

  • Start up;
  • User press the reload button;
  • Config was generated;
  • Target / task was added;
